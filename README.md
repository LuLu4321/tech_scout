# TechScout


## Requirements
- Python 3.10 or above
- 8GB memory

## Installation
### Update pip
`python.exe -m pip install --upgrade pip`

### Install packages
`pip install -r requirements.txt`

## Running

From your Python (virtual) environment, run Tech Scout:

`python tech_scout.py`
